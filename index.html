VFA Rock Lobster Status Reports



Welcome to the interactive stock status reports for the Victorian Rock Lobster fishery. The Victorian Fisheries Authority (VFA) has now made available an online interactive version of monthly and annual stock status reports. A pdf version of stock status reports are still available from the VFA website.

Stock status reports are available for both the Eastern and Western zones. View them here:

  • EAST ZONE ROCK LOBSTER FISHERY
  • WEST ZONE ROCK LOBSTER FISHERY

Features and Instructions:


The stock status reports for each zone have three main views. Use the Annual, Monthly, and Models tabs to select each view:

  • Annual: Displays fishery information for the most recently completed fishing season (Nov-Sept)
  • Monthly: Displays information on catch, effort, and total allowable catch (TAC) for the current fishing season, updated monthly
  • Models: Displays outputs from the Rock Lobster fishery assessment model for the most recently completed fishing season

Reports use interactive features for displaying information. Interactive plot features include dropdown selection boxes, detailed information display on mouse hover, selectable traces and click-and-drag zoom functions. These features are explained in more detail below:



Maps:

The fishery information displayed on the annual commercial catch by zone map can be changed using the dropdown menu above the map (catch, effort or cpue).

The area in view can be changed by:

  1. hold-clicking the mouse and dragging
  2. zooming in and out via the +/- buttons in top left.

Fishing zones are colour-coded according to the key in the lower right. In general, a darker shade represents a higher value. The zone name and exact value is displayed in the top right box.



Graphs/Plots:

The type of data displayed can be changed via dropdowns above the plots (where applicable).

When the mouse is hovered over the plot, a number of grey icons will appear in the upper right corner. These icons will allow you to:

  1. Save the current plot as a picture (png) file
  2. Zoom or pan
  3. Autoscale or reset the axes
  4. Change how data data is displayed on mouse hover.

When hovering the mouse over a specific point, line or bar, detailed information will "pop-up". You can change settings for this in the icon menu.

Zoom-in to view only specific years by hold-clicking the mouse and dragging a box over the data you want to focus on. Return to the full view by clicking "Autoscale" or "Reset axes" on icon menu.

Data display can also be turned on or off by clicking on the names in the legend (top right of the plot) e.g. in the Commercial catch history plot, clicking on the region name on right hand side will toggle the display of this region on or off.



Tables:

Tables can be sorted via any column header (alphabetically for text columns, numerically for number columns) by clicking on the column header.



Show/Hide Data:

Many plots have a Show/Hide Data button below them. On click this will open the data table behind each plot, which can be selected and copied if required. Clicking the "Show/Hide Data" button again will hide the data table.
